# UdacityP4
Udacity Data Visualization Project

## 概要  
本项目使用D3.js完成可交互式数据可视化，用于展示航班延误信息。
## 设计  
选定1987年航班信息数据，为实现快速加载，截取其中20000条数据的部分字段。分别展示在每周七天里机场的延误次数和每个月里机场在每天的延误次数，用户可选择按照每周还是每月显示。
